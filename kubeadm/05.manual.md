## 일상 점검 명령어

### Kubernetes 클러스터
```bash
kubectl get nodes
kubectl get pods -A
kubectl get svc -A
```

### Jenkins
```bash
kubectl get pods -n jenkins
kubectl logs -n jenkins deployment/jenkins -f --tail=100
```

### Argo CD
```bash
kubectl get pods -n argocd
kubectl get applications -n argocd
argocd app list   # CLI 사용 시
```

### 시스템 서비스 (노드에서)
```bash
sudo systemctl status containerd
sudo systemctl status kubelet
```
